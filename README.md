# DevOps Learning Projects

This repository contains my DevOps learning projects using AWS, Terraform, Kubernetes, and related technologies.

## Structure

- `terraform/` - Infrastructure as Code projects using Terraform
- `kubernetes/` - Kubernetes manifests and configurations
- `docker/` - Dockerfiles and container configurations
- `ansible/` - Ansible playbooks and roles
- `scripts/` - Utility scripts for automation
- `docs/` - Documentation and learning notes

## Getting Started

Each subdirectory contains its own README with specific instructions.

## Prerequisites

- AWS CLI configured with credentials
- Terraform installed
- kubectl installed
- Docker installed
- Git configured

## Learning Path

Track your progress and projects here as you learn DevOps practices.
